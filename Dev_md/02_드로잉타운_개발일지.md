# 02. 드로잉타운 (DrawingTown) — 개발일지

작업일: 2026-06-08 ~ 2026-06-09

---

## 프로젝트 개요

예비 일러스트레이터 · 인스타툰 작가를 위한 드로잉 학습 커뮤니티 사이트.
현직 작가의 유튜브 영상을 모아 소개하고, 챌린지와 멤버 커뮤니티 기능을 제공한다.

- **배포 URL**: https://kwonqbeen-prog.github.io/day06/
- **리포지토리**: https://github.com/kwonqbeen-prog/day06

---

## 기술 스택

| 항목 | 선택 |
|------|------|
| 프레임워크 | React 18 + Vite 5 |
| 스타일링 | Tailwind CSS v3 |
| 라우팅 | React Router 6 (HashRouter) |
| 배포 | GitHub Pages (GitHub Actions 자동 배포) |
| 폰트 | Pretendard (본문) · Fredoka (로고) |

> HashRouter 선택 이유: GitHub Pages는 서버 사이드 라우팅을 지원하지 않아 BrowserRouter 사용 시 새로고침 또는 직접 URL 접근 시 404 발생. HashRouter로 회피.

## 구현 기능

### 1. 컬러 팔레트 시스템 (5 팔레트 × 라이트/다크)

- CSS 커스텀 프로퍼티(`--c-primary`, `--c-secondary` 등)를 `:root[data-palette="N"]` 속성 선택자로 관리
- Tailwind `darkMode: 'class'` + 커스텀 컬러 토큰 연결
- `ThemeContext`로 팔레트·테마 상태 전역 관리, `localStorage` 영속화
- 5종 팔레트: 드림(보라/핑크) · 코튼(하늘/분홍) · 민트(민트/퍼플) · 피치(살구/핑크) · 로즈(로즈/보라)

### 2. 헤더 / 푸터

- 데스크탑 메가메뉴 + 모바일 슬라이드 패널
- 헤더 내 팔레트 원형 스위처 5개 + 다크/라이트 토글 버튼
- 로고: 팔레트 아이콘 SVG + 한글 '드로잉타운' (Fredoka 폰트)
- 푸터: 브랜드 설명 · 팔레트 선택 · SNS 링크 · 빠른 링크

### 3. 홈 (Hero 슬라이더)

- 5슬라이드 자동전환 (4.5s): 추천 영상 썸네일 3개 + 멤버 슬라이드 2개
- 영상 슬라이드: YouTube `maxresdefault.jpg` → 실패 시 `hqdefault.jpg` 폴백
- 멤버 슬라이드: 그라데이션 배경 + 프로필 이미지 우측 배치 (없으면 그라데이션만)
- **Liquid glass 효과**: `backdrop-blur-[7px]` 레이어를 전체 히어로에 씌우되, `mask-image: linear-gradient(to right, black 25%, transparent 72%)`로 왼쪽은 강하게 흐리고 오른쪽은 선명하게 유지
- 하단 인디케이터, 우하단 슬라이드 정보 표시

### 4. 추천 영상 페이지 (`/videos`)

- 3개 채널 탭 필터 (전체 / Studio MEI / illustrator윰마 / 은지 Eunji)
- 2열 그리드, 6개/페이지, 페이지네이션
- YouTube 실제 썸네일 사용, 호버 시 재생 버튼 오버레이

### 5. 참고 채널 페이지 (`/artists`)

- 3개 유튜브 채널 카드: Studio MEI · illustrator윰마 · 은지 Eunji Illustration
- YouTube 구독 버튼 (실제 채널 URL 연결)

### 6. 멤버 시스템

| 경로 | 기능 |
|------|------|
| `/members` | 멤버 4인 그리드 목록 |
| `/members/:id` | 개별 프로필 + 게시물 목록 |
| `/members/post` | 게시물 작성 폼 (멤버 선택·제목·내용·태그) |

- `MemberImage` 컴포넌트: `public/images/members/{id}.jpg` 로드 시도 → 실패 시 그라데이션 폴백
- 가상 멤버 4인: 수지(캐릭터/인스타툰) · 민하(수채화/자연) · 지노(디지털/판타지) · 아름(굿즈/아크릴)
- 샘플 게시물 8개 (`src/data/site.js`의 `posts` 배열)

### 7. 챌린지 · 커뮤니티 · 고객센터

- 챌린지: 진행 중/예정 카드, 참여 방법 섹션
- 커뮤니티: 공지사항 탭 + 갤러리 탭 (탭 전환)

## 트러블슈팅

| 문제 | 원인 | 해결 |
|------|------|------|
| `hover:shadow-primary/20` 빌드 오류 | Tailwind v3에서 CSS 변수 컬러에 opacity modifier(`/20`) 사용 불가 | `index.css` `.card`에서 `/20` 제거 |
| `Challenge.jsx` 빌드 오류 | `site.js`에서 `artists` → `channels` 리네임 후 import 미수정 | import 제거 및 정적 텍스트로 대체 |
| GitHub Pages 404 | `vite.config.js`의 `base`가 이전 프로젝트명(`/rest03/`)으로 설정됨 | `/day06/`으로 수정 |
| GitHub Actions 미실행 | 저장소 Actions 권한 비활성화 상태 | Settings → Actions → General → "Allow all actions" 활성화 |
| 프로필 이미지 미표시 | 이미지를 리포 루트에 업로드, 코드는 `public/images/members/` 참조 | `git mv`로 올바른 경로로 이동 |
